# Review 2026 Logs

This folder contains logs of all changes made during the review process for the 2026 edition of the book.

## Structure

- `logs/` - Individual chapter change logs
- `summary.md` - Overall summary of review progress

## Logging Changes

When making changes to any chapter, update the corresponding log file in `logs/` with:
- Date of change
- What was changed
- Why it was changed
- Who made the change (if applicable)

## Chapter Files

Each chapter has a corresponding log file named to match the chapter file name:
- `logs/00-intro-book.md` - The Book (Introduction)
- `logs/01-intro-logic.md` - Short Intro to Logic
- `logs/02-intro-propositional-logic.md` - Propositional Logic
- `logs/03-intro-modal-logic.md` - Modal Logic
- `logs/04-sat-solving-minisat.md` - SAT solving with MiniSat
- `logs/05-logic-programming-prolog.md` - Logic Programming with Prolog
- `logs/06-constraints-minizinc.md` - Constraints with MiniZinc
- `logs/07-smt-solving-z3.md` - SMT Solving and Z3
- `logs/08-temporal-logic-spin.md` - Temporal Logic with Spin
- `logs/09-epistemic-logic-smcdel.md` - Epistemic Logic with SMCDEL
- `logs/10-hoare-logic-dafny.md` - Hoare Logic with Dafny
- `logs/11-higher-order-logic-isabelle.md` - Higher-Order Logic with Isabelle
- `logs/12-type-theory-lean.md` - Type Theory with Lean
- `logs/appendix-*.md` - Appendices (e.g., `appendix-complexity-classes.md`)
