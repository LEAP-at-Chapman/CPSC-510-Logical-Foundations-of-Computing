# Cursor Agent Rules for Book Review

This folder contains rules and guidelines for the Cursor AI agent to help with reviewing and improving the book "Logical Foundations of Computing" on the `review-2026` branch.

## Context

- **Book**: Logical Foundations of Computing (CPSC-510)
- **Branch**: `review-2026`
- **Authors**: Written collaboratively with students
- **Goal**: Improve the book to a level where it can be proudly presented to colleagues, while retaining the student work and maintaining an appropriate standard (not requiring professional academic publication quality)

## Guidelines

See the individual rule files in this directory for specific guidelines.

## Important Style Guides

**CRITICAL**: The AI agent must follow these style guides located in `content/`:

- **`content/how-to-cite.md`** - Citation format and reference guidelines
- **`content/how-to-contribute.md`** - Writing style, organization, and contribution guidelines

These files contain the official style standards for the book and must be consulted when making any changes.
